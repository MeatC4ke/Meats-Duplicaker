# Meats-Duplicaker

A Unity Script made for instantiating/Duplicating Objects easier

HOW  TO USE:

Name: *Duplicate object per name*

Asset URL: *Duplicate object per url*

Position: *where the duplicate spawns*

Clone name: *Name of the duplicate*
